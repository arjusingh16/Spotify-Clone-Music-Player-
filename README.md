Spotify Clone

This project is a web-based Spotify Clone built using HTML, CSS, and JavaScript. It replicates the basic functionalities of a music player, including playing, pausing, and switching between songs. The project demonstrates front-end development skills such as layout design, DOM manipulation, and event handling.

---

Features

- Music play and pause functionality
- Next and previous song controls
- Dynamic album cover and background updates
- Responsive design for various screen sizes
- Organized asset structure for images and songs

---

Project Structure

Spotify-Clone/
│
├── covers/        # Contains album cover images
├── songs/         # Contains audio files
├── bg/            # Contains background images
│
├── index.html     # Main HTML file
├── style.css      # Stylesheet for layout and design
├── script.js      # JavaScript for functionality
├── logo.png       # Application logo
├── playing.png    # Image used for playing state

---

Technologies Used

- HTML5 – for structure
- CSS3 – for styling and layout
- JavaScript (ES6) – for interactivity and logic

---

How to Run the Project

1. Clone the repository:

git clone https://github.com/<your-username>/Spotify-Clone.git

2. Navigate to the project directory:

cd Spotify-Clone

3. Open the "index.html" file in any modern web browser.
